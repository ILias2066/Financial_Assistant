# Financial Assistant

Проект Financial Assistant предназначен для управления личными финансами с помощью Telegram-бота. Бот помогает пользователю управлять своим бюджетом, включая информацию о доходах, расходах, кредитах и других финансовых обязательствах.

## Структура проекта

- **main.py**: Основной файл приложения.
- **handlers/basic.py**: Обработчики команд для Telegram бота.
- **database/db.py**: Настройки базы данных (PostgreSQL).
- **Dockerfile**: Docker конфигурация для контейнеризации приложения.
- **docker-compose.yml**: Docker Compose конфигурация для запуска приложения и базы данных.
- **requirements.txt**: Список зависимостей Python.
